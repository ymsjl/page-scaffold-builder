# Page Scaffold Builder

Standalone copy of the PageScaffoldBuilder with a lightweight Fastify backend.

Workspace layout:

- apps/web: Vite React frontend
- apps/api: Fastify API
- packages/shared: shared types

Quick start:

```bash
pnpm install
pnpm dev
```

Useful commands:

- pnpm dev:web
- pnpm dev:api
